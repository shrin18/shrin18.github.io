What is Super-Optimisation ?

Randomised automated approach to find the optimal code sequence for loop-free sequence of instructions

STOKE Project

    ⇒ Optimiser for   

    x86-64 Instruction Set Architecture

        ⇒ Built on the MCMC Optimisation Algorithm

      Steps for Experiment

           1. Code compilation using g++ -O3 optimising options

      ⇒ Gives full Optimised Codes according to -O3 Optimisations of the GCC Compiler

      2. Testcase Generation

      ⇒ Randomised generation and backtracking

      3. Synthesize/search the code for Optimisation

      ⇒ Gives lowest cost verified rewrite as output code


      4. Rewriting the Binary for the optimised code

      ⇒ Replaces Binary Code by Optimised Version of the code

      ⇒ Can be verified by timeout
